프로젝트 개요

노션 데이터베이스와 연동하여 팀원들이 프로젝트와 일정을 관리할 수 있는 웹 기반 캘린더 제공

주요 기능

노션 데이터베이스 연동

  • • 프로젝트 데이터 가져오기
  • • 새로운 프로젝트 추가
  • • 기존 프로젝트 수정
  • • 프로젝트 삭제

웹 기반 캘린더 UI

  • • FullCalendar.js 기반 UI
  • • 드래그 앤 드롭 일정 관리
  • • 직관적인 일정 추가/수정
  • • 간편한 일정 삭제

실시간 데이터 동기화

  • • 자동 초기 데이터 로드
  • • 실시간 동기화 처리
  • • 주기적 데이터 새로고침
  • • 안정적인 데이터 관리
